## Usage:
1. Clone this repo <br>
```git clone https://github.com/Mayukhdeb/worm-embryo-classifier.git```
2. navigate into the folder <br>
```cd worm-embryo-classifier```
3. Run app.py <br>
``` python app.py```
4. Open this URL in your browser <br>
```http://127.0.0.1:5000/```
